# Disclaimer

This project is for education, interoperability research and owner-authorised maintenance of hardware.

Firmware variables control security, power, boot, and hardware-initialisation behaviour. Incorrect changes can prevent POST or boot, trigger drive-encryption recovery, invalidate support or warranty coverage, reduce security, corrupt data or require board-level recovery. No script in this repository can determine that a particular hidden setting is safe for a particular machine.

The maintainers do not warrant that:

* a Dell model is compatible;
* an offset remains valid after any firmware change;
* an undervolt is stable;
* a recovery method will work;
* a third-party binary is fit for use or legally redistributable;
* disabling a firmware lock is appropriate for a given security environment.

Use only on hardware you own or are explicitly authorised to service. Keep backups and a tested recovery plan. Do not use the toolkit to bypass controls on managed or third-party systems.
